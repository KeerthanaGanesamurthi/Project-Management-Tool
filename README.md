# Project Management tool
A MERN project to manage all your personal projects, might be used as portfolio as well. Built with MERN stack and Redux.


## Built With

- MONGODB
- EXPRESS JS
- REACT
- NODE JS
- REDUX
- REACT-BOOTSTRAP
- EXPRESS-VALIDATOR
- MULTER
- ESLINT
- GITHUB ACTIONS
- VSCODE
